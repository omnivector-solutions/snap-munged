# snap-munged

This snap provides the `munged` daemon, and the `munge`, `unmunge`, and `remunge` executables.


### Build
```bash
snapcraft cleanbuild
```

### Install
```
sudo snap install munged_0.1_amd64.snap --classic --dangerous
```
